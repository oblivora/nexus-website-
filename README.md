# Nexus Create — Main Website

Static home decor review & discovery website built with HTML/CSS/JS.

**Backend:** Supabase (read-only public queries)  
**Hosting:** Vercel  
**Live:** https://nexus-website.vercel.app

## Structure

```
pages/        → all website pages
assets/       → images, CSS, JS
templates/    → shared HTML partials
index.html    → root redirect → pages/
vercel.json   → Vercel routing config
```
